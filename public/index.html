MINERVA CHESS IS FINALLY HERE!!!

The epic six month adventure has finally concluded!! My application MINERVA CHESS is a fully functional chess engine! I programmed the engine to allow an average/beginner chess player to hang with it. At its core, the engine uses the alpha-beta search algorithm with various move ordering heuristics to choose the best move in a given position. My program also includes a multiplayer page, which allows users to create profiles and play against each other. Be sure to check it out!!!!
